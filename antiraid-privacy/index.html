Privacy Policy – Antiraid Extreme Backups

Effective Date: November 28, 2025

1. Data We Collect

Our bot collects only the data necessary to provide moderation and anti-raid features, including:

  • Message data: Temporarily analyzed to detect spam, flood, and self-bot behavior. Not stored long-term.
  • Presence data: Temporarily used to detect suspicious automated activity. Not stored long-term.
  • Server and user IDs: Used for moderation actions, logs, warnings, and notifications.

2. How We Use Your Data

Data is only used to:

  • Prevent spam, raids, and automated self-bots.
  • Provide moderation tools to server owners and moderators.
  • Enable bot features that interact with the community.

No data is sold, shared externally, or used for advertising.

Users can access this Privacy Policy by visiting this page at any time: this link.

4. Data Deletion Requests

Users may request deletion of their data by contacting us through one of the following:

5. Security and Retention

Temporary message and presence data is only kept in memory to detect automation. Server logs are stored temporarily and accessible only to moderators or server owners. No external backups or third-party storage is used beyond Discord.

We may update this policy to reflect new features or compliance requirements. Users will be notified through Discord or the website.
